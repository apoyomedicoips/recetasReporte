
# IPS Analytics Dashboard 2025

Dashboard interactivo para análisis de recetas médicas del IPS Paraguay.

## Instalación

1. Sube todos los archivos a tu repositorio
2. Ejecuta el script Python para generar datos
3. Abre https://apoyomedicoips.github.io/recetasReporte/

## Funcionalidades

- KPIs en tiempo real
- Filtros avanzados
- Gráficos interactivos
- Tablas exportables
- Tema oscuro/claro

## Datos

Los datos se generan desde archivos Parquet mensuales usando el script `procesar_todo_2025.py`.
